# Simple Python Calculator

A basic Python calculator that allows the user to perform up to **3 arithmetic calculations** in one run. Supports addition, subtraction, multiplication, and division, with error handling for invalid inputs and division by zero.

## Features

- Perform up to 3 calculations per run
- Supports `+`, `-`, `*`, `/` operations
- Handles invalid number inputs
- Prevents division by zero
- Alerts user if more than 3 calculations are requested

## How to Use

1. Clone or download this repository.
2. Run the Python file in your terminal or IDE:

```bash
python calculator.py
